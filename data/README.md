# Data

Normalized data area for future cleanup.

## Subfolders
- `raw/`: immutable source inputs
- `processed/`: derived datasets
- `external/`: third-party datasets or imports

## Current status
Historical project data has been migrated into this structure.

- `data/raw/covariates/`: phenotype and covariate files
- `data/raw/project-data/`: atlas and project data files
- `data/raw/events-recording/`: subject event files
- `data/processed/workspaces/`: saved `.RData` workspaces

`2 - materiale/` is kept as a compatibility path through links.
